    EchoMind

    Start the enterprise stack. It may take a few minutes before the app is ready — use Open in Services when it shows Ready.

    What this does (technical)

    Runs docker compose down then docker compose up -d in the enterprise folder in the background so the browser does not hit a proxy timeout. Then open the EchoMind URL from the result or from Services.

    Personaplex · Conversation

    Start the voice conversation server, then open the link when it appears below.

    What this does (technical)

    Starts moshi.server only (matches typical manual steps): cd repo → activate ~/personaplex-venv → temp SSL dir → python -m moshi.server --fp8 --ssl … --static client/dist in the background. No npm build, no pkill. Requires existing client/dist. Logs: /tmp/moshi-personaplex.log.

    Wi‑Fi hotspot

    Enable turns on the hotspot (and runs first-time setup if needed). Disable turns it off and restores normal Wi‑Fi. Safe stop stops the AP only (no NetworkManager undo). Both Disable and Safe stop set a session flag so if dgx-app restarts, the hotspot is not turned back on automatically until you click Enable or reboot.
